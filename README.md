# CreditWise Loan System

## Project Description
CreditWise Loan System is a supervised machine learning project that predicts whether a loan application is likely to be approved.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training
- Loan approval prediction

## Algorithms Used
- K-Nearest Neighbors (KNN)
- Logistic Regression
- Naive Bayes

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Note
The dataset (`loan_approval_data.csv`) is not included in this repository as per the assignment requirements. Please place the dataset in the project folder before running the notebook.
